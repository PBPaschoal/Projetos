Página Web para Empresa de Transporte e Logística Internacional

O projeto consiste no desenvolvimento de uma nova página web para uma empresa especializada em transporte e logística internacional. Com o objetivo de destacar os serviços oferecidos e proporcionar uma experiência de usuário intuitiva e informativa, a página foi projetada com um design moderno e funcionalidades interativas.

A página inicial apresenta um carrossel dinâmico exibindo diferentes modos de transporte, como navio, avião e trem, refletindo a ampla gama de serviços da empresa. Além disso, uma seção dedicada aos serviços oferecidos destaca os principais aspectos do transporte rodoviário, aéreo, marítimo, cargas projeto e cabotagem, cada um acompanhado de uma descrição detalhada e uma imagem ilustrativa.

Outro destaque do projeto é a seção "Quem Somos", onde a empresa apresenta sua missão, visão e valores, além de destacar sua abordagem inovadora e bem-sucedida nos negócios. Essa seção fornece uma visão mais profunda sobre a empresa e seus objetivos organizacionais.

Tecnologias Utilizadas:
HTML5 e CSS3: Utilizados para a estruturação e estilização da página web, garantindo um layout responsivo e agradável visualmente.
Bootstrap: Framework CSS utilizado para o desenvolvimento de componentes e estilos responsivos, agilizando o processo de criação e garantindo compatibilidade com diversos dispositivos.
UIKit: Framework front-end que fornece uma coleção extensa de componentes e estilos personalizáveis, contribuindo para a construção de uma interface moderna e dinâmica.
JavaScript (jQuery): Utilizado para adicionar interatividade à página, como o controle do menu de navegação em telas menores e animações.
