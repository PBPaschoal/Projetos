Simulador de Empréstimo Consignado - V06

Este projeto em Java é um simulador de empréstimos que permite aos usuários cadastrarem-se e acessarem o sistema para simular diferentes tipos de operações financeiras (é realizado localmente no programa Java, sem o uso de um banco de dados). Abaixo estão algumas das principais funcionalidades do projeto:

Cadastro de Usuário e Senha:
No início do projeto, os usuários podem se cadastrar fornecendo um nome de usuário e uma senha. Caso o usuário erre o limite máximo de tentativas, é necessário cadastrar-se novamente. Acesso ao Sistema:

Após o cadastro, os usuários podem acessar o sistema fornecendo suas credenciais. Escolha do Banco e Tipo de Operação:

Os usuários têm a opção de escolher o banco desejado e o tipo de operação que desejam simular: empréstimo novo ou refinanciamento de dívidas.
Simulação de Contrato:
Para empréstimos novos, os usuários podem simular com base no valor da parcela desejada ou no valor total do contrato, proporcionando flexibilidade.
Para refinanciamento de dívidas:
Os usuários fornecem o valor da parcela e o saldo devedor.

Recursos de Programação Utilizados:
O projeto foi desenvolvido em Java, utilizando conceitos de programação orientada a objetos, tais como:

  • Classes
  • Métodos
  • Encapsulamento
  • Sobrecarga
  • Palavra-chave "this"
  • Arrays
  • Estruturas condicionais
  • Estruturas de repetição
